# パブリックツール

自分用の公開ダウンロード置き場です。
外出先などで必要になったツールやスクリプトを置いています。

## VB6解析ツール

VB6プロジェクトの構成を確認するための簡易ツールです。
`.bas`、`.frm`、`.cls` などのファイル構成を確認する目的で使用します。

## ファイル

* `VB6_Inventory.ps1`
  解析処理を行うPowerShellスクリプトです。

* `解析実行_.bat`
  PowerShellスクリプトを実行するためのバッチファイルです。

## 使い方

1. 解析したいVB6プロジェクトのフォルダに、この2つのファイルを置きます。
2. 必要に応じて、バッチファイルまたはPowerShellスクリプト内の対象フォルダ指定を確認・変更します。
3. `解析実行_.bat` を実行します。
4. 出力された結果を確認します。
